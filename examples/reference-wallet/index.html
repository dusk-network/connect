Minimal injected wallet + dApp handshake

Connect Discovery Dario

This page injects a tiny Dusk wallet implementation through the raw browser discovery events, then consumes it through createDuskWallet().

It is meant to show wallet authors the smallest useful integration shape: announce, connect, react to provider events, and keep metadata stable.
